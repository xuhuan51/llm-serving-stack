# Experiment Reports

This directory stores reproducible experiment notes for the AI Infra Lab.

Use these reports to connect raw benchmark data with infrastructure conclusions:

```text
benchmark says what changed
monitor explains why it changed
diagnosis turns the evidence into an action
```

## Files

- `llm-inference-observability-template.md`: reusable report template for new experiments.
- `llm-inference-case-studies.md`: current vLLM benchmark and monitoring case studies.

## Report Rules

- Record exact commands and service parameters.
- Keep benchmark and monitor evidence together.
- Separate observation from interpretation.
- Always end with a tuning action or capacity-planning conclusion.
